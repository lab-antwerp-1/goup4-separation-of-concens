# Handlers

## removeItemHandler

- Entry point: user interaction. To remove a specific item from the list.
- It `import` from:
  - data.js
  - `logic/remove-item.js`
- It `export` to `/listeners/remove-item.js`.

## removeListHandler

- Entry point: user interaction. To remove a specific list.
- It `import` from:
  - data.js
  - `logic/remove-item.js`
- It `export` to `/listeners/remove-item.js`.
